Tree23.h - c++ library

template <class T> class Tree23;

Tree23 is a class implementing ordered set as (2,3)-tree.

This library requeres c++98 standard.

Demo

#include <iostream>
#include "Tree23.h"

int main() {
    float inp[5] = {50.5, 10.7, 30.2, 20.2, 40.4}; 
    float fnd[5] = {50.5, 30.2, 40.4, 3.14, 2.72};
	
    Tree23<float> tr;

	for (int i=0; i<5; i++) {
		tr.insert(inp[i]);
		std::cout << inp[i] << " inserted";
        	std::cout <<" (size: " << tr.size() << ") " << std::endl;
	}
	
	for (int i=0; i<5; i++) {
		if (tr.find(fnd[i])) {
			std::cout << fnd[i] << " found" << std::endl;
		} else {
			std::cout << fnd[i] << " not found" << std::endl;
		}
	}
	
	for (int i=0; i<5; i++) {
		tr.erase(fnd[i]);
		std::cout << inp[i] << " erased";		
		std::cout <<" (size: " << tr.size() << ") " << std::endl;
	}
	
	tr.clear();
	
	if (tr.empty()) {
		std::cout << "set  clear";
		std::cout << " (size: " << tr.size() << ") " << std::endl;				
	}
	
    return 0;

constructor

Tree23::Tree23();

Inicialize set structure.

Time complexity is constant.

destructor

Tree23::~Tree23();

Deallocate all used pointers and volues.

Time complexity is linear in actual size.

Insert

void Tree23::insert(const T &val);

Insert value into the set. Value is passed by reference and will be copied into class.

Time complexity is logarithmic in size.

Find

bool Tree23::find(const T &val);

Find out if value is in the set. Value is passed by reference.

Time complexity is logarithmic in size.

Erase

void Tree23::erase(const T &val);

Delete value from the set. Value is passed by reference.

Time complexity is logarithmic in size.

Size

inline unsigned Tree23::size();

Returns number of values in the set.

Time complexity is constant.

Empty

inline bool Tree23::empty();

Returns true if size of the set is equal to zero, false otherwise.

Time complexity is constant.

Clear

void Tree23::clear();

Delete all values from set and call new contructor.

Time complexity is a time complexity equals to complexity of constructor and destructor.
